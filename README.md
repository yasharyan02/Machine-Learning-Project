This project aims to detect fraudulent credit card transactions using machine learning techniques. Due to the highly imbalanced nature of fraud datasets, 
special preprocessing and modeling techniques are applied to improve detection accuracy.
Dataset Download link: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
Project Workflow
1️. Data Preprocessing
Data cleaning
Feature scaling
Handling imbalanced dataset using:
SMOTE
Undersampling

2️. Exploratory Data Analysis (EDA)
Fraud vs Non-Fraud distribution
Correlation matrix
Data visualization

3️. Model Building
Implemented models:
Logistic Regression
Decision Tree
Random Forest
Support Vector Machine (SVM)

4️. Model Evaluation
Evaluation metrics used:
Accuracy
Precision
Recall
F1-Score
ROC-AUC Curve
